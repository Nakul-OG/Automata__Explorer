📖Operator ReferenceSYNTAX
ab
Concatenation
Two expressions side-by-side: first then second in sequence.
ab → {ab}
+
Union
Matches either the left or the right expression (logical OR).
a+b → {a,b}
*
Kleene Star
Zero or more repetitions. Includes the empty string ε always.
a* → {ε,a,aa,…}
|
Kleene Plus
One or more repetitions. Like star but excludes ε.
a| → {a,aa,…}
?
Optional
Zero or one occurrence — makes the preceding atom optional.
ab? → {a,ab}
.
Dot — Any
Matches any single character from a–z, 0–9.
a. → {aa,ab,…}
[]
Character Class
One char from the set. Use a-z for ranges; ^ to negate.
[a-c] → {a,b,c}
ε
Empty String
Represents a string of length zero. Type ε or leave blank.
ε → {""}
